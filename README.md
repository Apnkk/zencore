# Zencore

A lightweight Windows application for system and application optimization.

![.NET](https://img.shields.io/badge/.NET-8.0-512BD4?style=flat-square&logo=dotnet)
![Windows](https://img.shields.io/badge/Windows-10%2F11-0078D4?style=flat-square&logo=windows)
![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)

## Features

- **Performance Optimization** — Boost application performance with one click
- **System Tweaks** — Apply proven system-level optimizations
- **Lightweight** — Minimal resource usage, runs silently in background
- **License System** — Secure key-based activation with online verification

## Installation

1. Download the latest release from [Releases](../../releases)
2. Extract the archive
3. Run `Zencore.exe`

## Requirements

- Windows 10/11
- .NET 8.0 Runtime

## Building from Source

```bash
git clone https://github.com/Apnkk/zencore.git
cd zencore
dotnet build
```
